# 2024-12-15 - React Native Mobile App & Auth Implementation

## Özet
React Native (Expo) mobil uygulaması oluşturuldu. Auth ekranları, dashboard, job details ve modal'lar eklendi. User isolation bug'ı düzeltildi.

---

## Yeni Özellikler

### 📱 Mobil Uygulama (Wagetracker.Mobile)
- **Expo projesi** TypeScript template ile oluşturuldu
- **Navigation**: React Navigation 7 ile auth/main flow ayrımı
- **State Management**: Zustand stores (auth, jobs, entries)
- **API Layer**: Axios client with token interceptors

### 🔐 Authentication
- `LoginScreen` - Email/password ile giriş
- `RegisterScreen` - Yeni hesap oluşturma
- `authStore` - Token yönetimi (SecureStore + localStorage fallback for web)

### 📊 Dashboard
- Bento grid layout (Today card, Earnings card, Hours card)
- Job listesi (`JobCard` component)
- **Profil butonu** (sağ üst) + dropdown menü ile çıkış
- Pull-to-refresh
- FAB ile yeni job oluşturma

### 💼 Job Details
- Weekly grouping görünümü
- Overtime indicators (turuncu badge)
- Entry silme
- `AddEntryModal` - Total hours veya start/end time modları

### 🎨 UI Components
- `Button` - primary/secondary/ghost variants
- `Card` - default + gradient variants (earnings/hours)
- `Input` - with label, error, icon support
- `CreateJobModal` - Job oluşturma formu

---

## Bug Fixes

### 🐛 User Isolation Bug (KRİTİK)
**Sorun:** Tüm kullanıcılar aynı veriyi görüyordu  
**Sebep:** Controller'larda `GetUserId()` development modunda hardcoded `1` döndürüyordu  
**Çözüm:** 3 controller'dan bypass kaldırıldı:
- `DashboardController.cs`
- `JobsController.cs`
- `EntriesController.cs`

### 🌐 Web Compatibility
- `expo-secure-store` web'de çalışmıyor → `localStorage` fallback eklendi
